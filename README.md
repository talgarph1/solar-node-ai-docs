## Project Overview

**Solar-Battery Node AI** is an integrated autonomous monitoring system for solar-powered space missions, combining:
- Machine learning-based battery health prediction (92% accuracy)
- Computer vision anomaly detection (<100ms inference)
- Edge AI processing on dual STM32 architecture
